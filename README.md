# Qivo - Quito en Movimiento

**Qivo** es una aplicación web para reservar traslados seguros en Quito y hacia/desde el aeropuerto, con conductores confiables y atención personalizada.

## Características
- Modal responsivo y centrado
- Branding institucional y degradados
- Formulario dinámico según servicio
- Integración con WhatsApp
- Información de vehículo y conductor
- Optimizado para SEO y dispositivos móviles

## Instalación y uso
1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.
3. Personaliza imágenes y datos en `app.js` según tus conductores.

## Estructura
- `index.html`: Estructura principal y modal
- `styles.css`: Estilos y responsividad
- `app.js`: Lógica de UI y WhatsApp
- `Images/`: Logos e iconos

## SEO y Accesibilidad
- Meta etiquetas optimizadas
- Imágenes con `alt` descriptivo
- Responsive para móviles y escritorio
